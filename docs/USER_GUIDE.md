## Understanding NFT Cards

Each NFT card displays:
- **Image**: The digital artwork
- **Title**: Name of the NFT
- **Description**: Brief description of the artwork
- **Artist**: Creator's name and profile picture
- **Price**: Cost in ETH (Ethereum)
- **Status**: "Available" badge for listed NFTs

## Wallet Connection

**Current Status**: The "Connect Wallet" button is visible in the header.

**What It Does**: 
- Allows you to connect your cryptocurrency wallet
- Required for purchasing NFTs
- Enables blockchain transactions

**Note**: Full wallet functionality requires ThirdWeb SDK integration.

## Search Functionality

The marketplace search helps you find NFTs by:
- **Title**: Search by NFT name
- **Description**: Find keywords in descriptions
- **Artist**: Search by artist name

Simply type in the search box and results update automatically.

## Understanding Blockchain Details

On NFT detail pages, you'll see:

**Token ID**: Unique identifier for the NFT on the blockchain
**Contract Address**: Smart contract that manages the NFT
**IPFS**: Decentralized storage link for the artwork
**Created Date**: When the NFT was minted
